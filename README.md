## 这是在联创新人实习期写的第二个新人任务
## 这是一个V2EX客户端

相当于一个网络爬虫吧.....,.

### 主要实现的功能

- 异步加载最新、最热主题并展示
- 可查看每个帖子的详细信息、他人回复等
- 查看回复实现上拉刷新、下滑加载更多功能
- 回复内容实现富文本
- 可查看所有节点、每个节点详细信息、用户信息
  这个任务总计用时**两周**，主要难点在于网络异步加载以及使用正则表达式筛选网页内容、回复内容的富文本实现

### 主要使用技术

- 网络异步加载机制
- Jsoup库的使用
- 正则表达式筛选网页内容
